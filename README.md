# lsh_cpy

## 
lshというリポジトリの機能を拡張したもの
簡易的な自作シェル

## 使い方
```bash
git clone https://github.com/YamasouA/lsh_cpy.git
gcc lsh_cpy.c
./a.out
```

## 対応コマンド
cd
ls
cat
help
exit
cp
